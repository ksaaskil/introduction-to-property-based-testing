# erlang_targeted_pbt

## Instructions

Build

    $ rebar3 compile

Run properties with [`rebar3-proper`](https://github.com/ferd/rebar3_proper/) plugin:

    $ rebar3 proper

Create new properties:

    $ rebar3 new proper foundations
    ===> Writing test/prop_foundations.erl

See rebar3-proper plugin help:

    $ rebar3 help proper
